# Hospital Management System

## Overview

Hospital Management System is a web-based application developed using Spring Boot, Hibernate, MySQL, HTML, CSS, and JavaScript. The system streamlines hospital operations by providing separate login modules for doctors and patients, appointment management, and efficient record handling.

## Features

### Home Page

* Responsive landing page
* Navigation bar with:

  * Home
  * About
  * Doctor Login
  * Patient Login
  * Contact

### Doctor Module

* Secure Doctor Login
* View Patient Details
* Manage Appointments
* Update Patient Records
* View Doctor Dashboard

### Patient Module

* Secure Patient Login
* Patient Registration
* View Personal Profile
* Book Appointments
* View Appointment Status

### Contact Page

* Contact Information
* User-Friendly Interface

### About Page

* Hospital Information
* System Overview

## How to Run the Project

1. Clone the repository.
2. Import the project into Spring Tool Suite (STS) or Eclipse.
3. Configure MySQL Database.
4. Update database credentials in `application.properties`.
5. Run the Spring Boot application.
6. Open:
   `http://localhost:8080`
